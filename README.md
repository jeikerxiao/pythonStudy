# 1.movieSpider

这个项目是一个爬取豆瓣电影排名前250部电影的简介。

项目使用Scrapy爬虫框架。

IDE推荐使用PyCharm。

## 运行
在PyCharm中，运行 entrypoint.py 文件。

### tag v0.0.1

输出排名前250位的电影名。

